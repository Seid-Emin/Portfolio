Original Movie Collections Website

2020

My attraction to
movies and tv series

As I move forward in my web development career, i decided to build a webside for movie and TV fans. To accomplish it i decided to use The Movie Database (TMDb) as it is a community built movie and TV database.

Another aspect was to make it possible for any user to create its own movie database like collections and keep track of it. Therefore i found handy to implement firebase and firestore API to accomplish the functionality.

The Tech Behind
The Website

With this project I wanted to create a Website that combined great design, a solid build foundation and created a unique and premium experience. As I grow as a developer i wanted to try and create my own design. For this reason this is a original website.

Along with a powerful Javascript framework ( React.js ) and its state manager Redux i decided to use clean CSS. Also, Git and Github, to ensure my versioning and documentation is readily accessible.

A Mobile friendly approach

With a vast majority of users visiting websites through their smart phones, having a mobile friendly website is crucial for every company. This project has allowed me to hone in my responsive design and development skills creating a more accessible and native experience for our users.
